# Nantes Finance

Site vitrine M&A pour la région de Nantes, prêt à être déployé sur Vercel via GitHub.

## Déploiement

1. Pousser ce dépôt sur GitHub.
2. Importer le projet dans Vercel.
3. Laisser Vercel détecter un site statique à la racine.

## Contenu

- Landing page en français
- Positionnement M&A Nantes / Pays de la Loire
- Design responsive
- Aucun framework requis
